# rtl8181-sdk
Realtek SDK/Toolchains for RTL8181 (MIPS)

## Contents

This contains the SDK (toolchains) for rtl8181 based systems. It also includes a few extra files, like bootloader code, firmware, Linux kernel source code, etc.
